# Code of Conduct

SciPhys OS is a technical community for experimental science infrastructure.

We expect contributors to:

- be respectful and constructive;
- distinguish evidence from speculation;
- welcome questions from students and early-career researchers;
- avoid personal attacks, harassment, or exclusionary behavior;
- protect private lab data and unpublished research.

Unacceptable behavior includes harassment, discriminatory language, deliberate
publication of private data, and repeated bad-faith disruption.

Report concerns privately to the project maintainers through GitHub.
