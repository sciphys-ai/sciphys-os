# Security Policy

SciPhys OS handles scientific files that may contain private lab metadata.

Please report security issues privately through GitHub security advisories when
available, or contact the maintainers before publishing details.

Do not open public issues that include:

- private sample or project names;
- unpublished experimental results;
- institutional file paths or operator names;
- credentials, API keys, or tokens;
- files that a lab has not approved for public release.

For parser bugs, prefer minimized and anonymized fixtures.
